# 多模态视觉智能安防监控系统

## 项目简介
针对多算法模块割裂、算力消耗高的问题，搭建一体化智能安防监控平台，实现多算法统一调度与高效推理。

## 技术栈
- Python / PyTorch
- CUDA / GPU 加速
- 模型轻量化（FP16 半精度推理）
- 可视化终端开发

## 核心优化成果
| 指标 | 优化前 | 优化后 |
|------|--------|--------|
| 推理帧率 | baseline | 提升 15-30 FPS |
| 显存占用 | baseline | 减半 |
| 硬件损耗 | 高 | 大幅降低 |

## 我的职责
- 独立设计全栈架构
- 自研 GPU 加速方案
- 完成模型轻量化与半精度推理优化
- 开发可视化终端
- 输出标准化运营报表

## 项目亮点
- 独立完成从架构设计到性能优化的全流程
- 推理性能提升显著，具备实际部署价值
